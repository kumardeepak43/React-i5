You simple pass in a function and an array of inputs and useMemo will only recompute the memoized value when one of the inputs has changed. Use of useCallback: useCallback will return a memoized version of the callback that only changes if one of the dependencies has changed. This is useful when passing callbacks to optimized child components that rely on reference equality to prevent unnecessary renders.
